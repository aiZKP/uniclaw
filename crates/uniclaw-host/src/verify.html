Uniclaw Receipt Verifier

Cold-verify a Uniclaw receipt in your browser. Nothing leaves this page.

Paste a receipt JSON below (for example, what you get from GET /receipts/<hash>) and click Verify. This page reconstructs the canonical body and runs an Ed25519 signature check against the receipt's embedded issuer key — entirely client-side using the browser's native crypto.subtle. There is no server-side verification.

⚠ Your browser does not appear to support Ed25519 in crypto.subtle. Try Chrome 113+, Firefox 130+, or Safari 17+. You can still inspect the receipt fields below — just not verify the signature here. The standalone uniclaw-verify CLI works on every platform.
See the project on GitHub for the receipt format spec (RFC-0001) and the offline command-line verifier (uniclaw-verify).
